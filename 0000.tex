\subsection{§000 序章 three years ago. (2015)}

% この日、ネバダのグルーム・レイクでは、アメリカが威信をかけて建設した巨大な加速器が稼働していた。
今天，在\ruby{内华达州}{Nevada State}的\ruby{古鲁姆湖}{Groom Lake}，这座为了彰显美国威信而建造的大型加速器开始运转。

% 地下１５０ｍに建設された、グルームレイクとボールド山にまたがる周囲１２０Ｋｍの大型加速器が、余剰次元の確認を行うべく出力を上げていた。
这座大型加速器位于地下150m处，横跨古鲁姆湖与\ruby{秃顶山}{Bald Mountain}，周长达到120km。为了验证\ruby{高维空间}{Extra dimensions}的存在，加速器正逐渐提高着输出功率。

% 衝突エネルギーがＬＨＣを遥かに超えたところで、数多くの粒子を衝突させたことが、計測器を通してメモリに記録され、結果がモニタに表示された。\\
监视器上显示着，侦测器记录下的，在远超\ruby{大型强子对撞机}{Large Hadron Collider}的能量下，无数的粒子对撞的结果。\\

% 「……マイクロブラックホールの生成を、確認しました！」\\
「……确认到\ruby{微型黑洞}{Micro black hole}的生成了！」\\

% 一斉にあがる歓声が、新しい理論が証明された瞬間を象徴していた。\\
众人一起发出的欢呼声，昭示着新的理论得到了实验的证明。\\

% 「タイラー博士、やりましたね！」\\
「成功了呢，\ruby{泰勒}{Tylor}博士！」\\

% この実験の責任者だった、セオドア＝ナナセ＝タイラー博士に、まわりの科学者が握手を求めて足早に近づいてくる。\\
四周的科学家们，都为了与这次实验的负责人\ruby{西奥多}{Theodore}·\ruby{七濑}{Nanase}·\ruby{泰勒}{Tylor}博士握手而聚了过来。\\

% 「やったな！　テッド！」
「成了啊！泰迪！」

% 「やめろよ、そのしゃべり出しそうなぬいぐるみみたいな呼び方は」\\
「别这么叫我啦，搞得跟那个会说话的玩偶似的」\\

% タイラーが笑いながら彼の手を握る。栄光の時間だった。\\
泰勒一边笑着一边与对方握手。这是他光荣的一刻。\\

% それを尊敬の眼差しで見つめていた若い科学者は、ひとしきり興奮を共有した後、ふとモニタに目を向けた。
而以尊敬的眼神望着博士的年轻科学家，沉浸在兴奋的气氛中一会之后，不经意间看了一眼监视器。

% 実験成功後もコンピューターは自分に与えられた仕事を淡々とこなしていた。
哪怕实验成功，计算机依旧默默地完成着自己的任务。

% フェムトセカントで積み上がるその情報は、プログラムに従って適切に処理され……信じがたい結果をモニタに映し出していた。\\
每一飞秒\footnote{飞秒：femtosecond，10^-15秒}都在不停积累的数据，经过程序的处理后……化作令人难以置信的结果显示在监视器上。\\

% 「な、タイラー博士！」\\
「什，泰勒博士！」\\

% 彼が思わずあげた、悲鳴に近い呼びかけは、まわりの注目を集めるのに充分だった。\\
他嘴中不经意间发出的，近乎悲鸣的喊声，吸引了当场所有人的注意力。\\

% 「どうしました？」
「怎么了？」

% 「マ、マイクロブラックホールが……消滅していません！」\\
「微、微型黑洞……没有消失！」\\

% そんな馬鹿な。そこにいた誰もがそう思った。
怎么可能。当场的所有人都这么想到。

% 理論が正しければ、ホーキング輻射によって、一瞬で蒸発してなくなるはずだ。生成に使われた質量は、たかだか陽子の質量に過ぎない。\\
如果理论是正确的，那么依据霍金辐射效应，黑洞将在瞬间蒸发。毕竟，生成这个黑洞，才使用了不过一个质子的质量。\\

% 「空間でいくつかのＭＢＨが高速に運動しています！　まるで……まるで、何かの力場に囚われているみたいだ……」\\
「空间里有几个微型黑洞正在高速运动！就好像……好像被某种力场束缚住一样……」\\

% ◇◇◇◇◇◇◇◇\\
\sqsplit\\

% 最初は量子レベルの空間の歪みに過ぎなかった。
一开始，空间的扭曲不过只到量子尺度。

% 刹那の時間の中で発生した歪みを、そ(・)れ(・)は、望みを叶えるための千載一遇のチャンスだと捕らえた。
可\textbf{那东西}，却抓住了这千载一遇的机会——刹那间的扭曲。

% そうして、そ(・)れ(・)は、その歪みをつかまえると慎重にエネルギーを加え、拡大していった。\\
\textbf{那东西}为了抓住这扭曲，慎重地向其注入能量，使其增大。\\

% ◇◇◇◇◇◇◇◇\\
\sqsplit\\

% 『固定された力場に、なにか巨大な質量が……なんだ、これは？！』\\
『在固定的力场中，有个质量巨大的东西……这是什么？』\\

% スピーカーから、誰かの悲鳴のような声が上がると同時に、モニターが白い光に覆われ、暗転して映像は終了した。\\
正在扬声器传出不知是谁近乎悲鸣的高喊后，显示器突然化作全白，然后变黑，视频戛然而止。\\

% 「それで全部かね？」\\
「这就是全部？」\\

% チェスターバリーの見事なスリーピースに身を包んだ、神経質そうな男が、足を組み直してそう尋ねた。\\
身着一套精致的\ruby{切斯特·巴里}{Chester Barrie}西装，显得有点神经质的男人，站定了脚这么问道。\\

% 「はっ、グルーム・レイク空軍基地で行われた加速器実験の地上コントロールで記録された映像はこれだけです」
「报告，这就是在古鲁姆湖空军基地进行的加速器实验的地上控制台中记录的所有视频了」

% 「つまり地上コントロールは無事なんだな？　電力供給用に建設された原発も」\\
「也就是说地上控制台没有危险咯？还有供应电力的核电站」\\

% 男の頭には、スリーマイルのトラブルがよぎっていた。いかにネバダとはいえ、あれの二の舞は御免だ。\\
男人的脑中，闪过了三哩岛的那次核泄漏事故。就算这里是内华达州，他也不想这件事重演。\\

% 「地上には大きな影響は出ていません。連絡が取れなくなったのは、加速器が設置されていた地下のみで、原発は無事です」
「对地上没有造成特别大的影响。失去联系的只有建造了加速器的地下，核电站工作正常」

% 「発生したＭＢＨは？」
「产生的微型黑洞呢？」

% 「わかりません。が、それがどうであれ、拡大して地球を飲み込むなどと言うことは考えられません」\\
「现状不明。但认为不会导致黑洞扩大将地球吞噬的的情况」\\

% 男はそれを聞いて、安心するように頷いた。\\
男性听到这句话后，安心地点了点头。\\

% 「地下への救出は？」
「地下人员的救援呢？」

% 「最初は基地の隊員で行われました」
「一开始我们派出了基地的队员」

% 「エレベーター類は完全に停止して動かなかったため、ボールド山の西側にあるポイント３の非常階段を利用して侵入したのですが……」\\
「由于电梯已经完全停止工作，我们从秃顶山西侧位置三处的紧急楼梯前往了地下……」\\

% 報告者は、モニタに静止画を映し出した。\\
报告者将几张照片显示在屏幕上。\\

% 「……なんだね、これは？　ハリウッドの新作か何かか？」\\
「……这是什么东西？好莱坞的新片子么？」\\

% そこには、青みがかった肌をした、恐ろしい顔の人型の何かが映っていた。\\
屏幕上，显示着一个全身青绿，面容狰狞的人形的东西。\\

% 「身長は10フィート以上あります。突入した部隊が、最初に出会った生命体です」\\
「身长估计有10英尺以上。这是突击部队最先遭遇的生命体」\\

% もしもそれがファンタジー映画だったら、きっとトロルやオーガと呼ばれていたに違いない。\\
要这是个幻想类视频，这东西的名字肯定是\ruby{洞穴巨人}{Troll}或者\ruby{食人魔}{Orge}吧。\\

% 「反射的に発砲した先頭の２名が犠牲になりました。GAU-5A ASDWでは豆鉄砲のようなもので、M855A1弾は、まるで効果がなかったそうです」
「最头上的两名队员反射性的开炮，不幸牺牲。GAU-5A ASDW就像玩具枪一样，M855A1弹几乎无法造成伤害」

% 「彼らは、フォボスとダイモスの間で瞬間移動装置の実験でも行っていたのか？」\\
「那些家伙难不成还做了\ruby{火卫一}{Phobos}和\ruby{火卫二}{Deimos}间的瞬移装置的实验么？」\\

% 唖然とした男は、若かりし頃夢中で遊んでいたゲームの設定を思わず口走ったが、すぐに首を振って、今できるいくつかのことを指示した。\\
震惊到不知说什么好的男性，不轻易间便讲出了年轻时玩的游戏\footnote{这里指id Software于1993年12月10日发布的游戏《毁灭战士》}里的设定。然后他马上摇了摇头，开始指示起现在能做的事情」\\

% ◇◇◇◇◇◇◇◇\\
\sqsplit\\

% その有機体は、確かになにかの知的活動を行っているようだった。
那个有机体，确实有进行了一些智力活动的迹象。

% 微弱で複雑な電流を絶えず発生させていた器官は、そ(・)れ(・)に、様々なことを示唆した。
对那不停地产生着微弱而复杂的电流的器官，\textbf{那东西}教唆了各种各样的事情。

% 刹那と永遠の間(はざま)で、そ(・)れ(・)は喜びにうちふるえると、莫大なリソースを解放した。自らの望みを叶えるために。\\
在那刹那与永恒的夹缝中，喜不自胜的\textbf{那东西}，释放出了巨量的资源——为了实现自己的愿望。\\

% この日、ネバダのグルーム・レイクの地下１５０ｍに、後にザ・リングと呼ばれるようになる最初のダンジョンが誕生した。\\
就在这一天，内华达州，古鲁姆湖地下150m处，后来被称做「\ruby{原初之环}{The Ring}」的迷宫，诞生了。\\
