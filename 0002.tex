% 「いや、これ、自然に、地割れに飲まれたんだよね？　俺関係ないよね？　弁償とか絶対無理――」\\
「呃，这个，是本来，就会掉进裂缝里的哟？跟我没关系的哟？要我赔是绝对赔不起——」\\

% 俺は冷や汗をかきながらそれを見ていた。
我一边全身冒着冷汗一边看着眼前的情景。

% どうせ頭のてっぺんから足の先まで雨でぐしょぐしょなのだ。今更冷や汗くらいどうってことないさと我ながら訳の分からないことを考えていたが、いつまで経っても鉄筋がぶつかる音が聞こえてこなかった。\\
反正我现在已经从头顶湿到脚跟了啦。再怎么流冷汗也不会怎么样——我脑子想着自己都觉得意义不明的事情，可钢筋着地的巨响却一直没有传来。\\

% 落ちたこと自体がなにかの間違いなんじゃ、と、地割れの脇をまわって近づいたとき、地の底から不気味な声のようなものが響いて来て、ぐらりと揺り返しが襲ってきた。\\
我正想着钢筋应该是掉下去了吧，走上前去想要确认，这时地底突然传来令人毛骨悚然的声音，接着便是一阵余震。\\

% 「なっ！」\\
「什！」\\

% そうして、体の中から何かに突き上げられるような感じが連続で起こり、それが終わると、目の前に虹色の綺麗な珠(オーブ)が現れた。
然后，我感觉身体里有什么东西正不停地被向上托起，等这感觉结束后，眼前出现了一颗闪烁着七彩光芒的\ruby{珠子}{orb}。

% それを見た俺は、思わず頭の中で落下距離を計算して逃避した。直径４ｃｍで、１０ｍくらいの鉄が自由落下したとすると、丁度１５秒くらいは経っていたから。……１０００ｍは越えている。\\
看见此情此景，我不自觉地计算起钢筋下落的距离，然后开始逃避起了现实。从直径4cm长10m左右的铁开始自由落体，已经过去了大约15秒。……已经过1000m了。\\
