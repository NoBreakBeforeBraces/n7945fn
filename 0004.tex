\subsection{§004 未登录技能与世界第一}

% オーブを掴んでバッグに入れると、俺はすぐにその場を立ち去った。
我抓住宝珠把它装进包里后，便匆匆离开了现场。

% なにしろトレーラが斜めに半分土に埋まっているのだ。穴はいつの間にか無くなっていたが、亀裂はトレーラーを飲み込んだまま残っていた。\\
毕竟这卡车已经斜着半截埋进土里了。落穴虽然不知在什么时候就消失了，但卡了辆卡车的裂缝依旧保持了下来。\\

% すでに全身ぐしょぐしょだったし、スキルオーブをあちこち持ち歩くのも嫌だったから、すぐに会社に早退の連絡を入れた。
我已经被淋了个落汤鸡，再说我也不打算带着技能宝珠东跑西跑，便立马联系公司早退。

% 電話の向こうで、榎木課長が俺を罵倒していたが、はい、はい、と機械的に相づちを打って電話を切った。\\
虽然电话那头，榎木科长不停地在骂着我，但我全部以机械的「是」回答应付了过去。\\

% そうして１時間後。俺は自分の部屋で、シャワーから出ると、年中ベッドの横に置いてある、コタツの前に腰掛けていた。\\
一小时后。我从家里的浴室洗完澡出来，然后在一年四季都摆在床边的被炉旁坐了下来。\\

% 「さてさて、これ、いくらくらいになるんだろうな」\\
「哎呀哎呀，这玩意能卖个多少嘞」\\

% オーブに触れると、その名称が分かる。
只要碰一下宝珠，就能知道其名字。

% その下にある謎の数字――オーブカウントと呼ばれていた――は、どうやら発見からの経過時間を表しているらしく、数値が 1436 になったあたりで消滅することがすでに知られていた。\\
在名字下面有一个迷之数字——据说是叫宝珠计时——似乎表示的是从发现起经过的时间，现在已经知道当这个数字变成1436的时候便会消失。\\

% 「メイキング / 0074 ね。５月の王ってのは、なんか凄そうだな。農業関係かな？」\\
「\ruby{MayKing}{メイキング}\footnote{这里原文为片假名，故导致了男主的错误理解} / 0074 啊。五月之王，听起来挺厉害的嘛。莫非跟农业有关？」\\

% 俺はノートＰＣを立ち上げると、ＪＤＡのオーブ購入リストにアクセスして、メイキングと入力してみた。が、結果は「なし」だった。\\
我开开笔记本，打开了JDA的宝珠收购列表，把「メイキング」输了进去搜索——然后，画面上就显示了一个「无结果」。\\

% オーブ購入リストは、さまざまな会社や組織や個人が、特定のオーブをいくらで購入したいという希望を記したリストで、発見者はそれに応募することでＪＤＡを介した取引が行われる。\\
宝珠收购列表是各种各样的公司、组织或者个人，对某个特定宝珠需求的列表，宝珠的发现者可以借此找到买家，经JDA介绍完成交易。\\

% なにしろ１日しか猶予がないのだ。
毕竟犹豫时间只有一天。

% いつ発見されるのかもわからないし、普通の店のように並んでいる商品を選んで購入する、なんてことは不可能だった。もちろんオークションを行う時間もないため、購入希望者との直接対話による売買が普通だった。\\
既不知道什么时候会出现，也没法像普通的商品一样供人挑选。自然，也没有举行拍卖的时间，因此一般都是发现者和求购者直接面谈完成交易。\\

% 「仕方ない。じゃ、どんな機能かだけでもチェックして――」\\
「没办法。那至少先确认下有什么用吧——」\\

% そう呟いた俺は、ＪＤＡのスキルデータベースにアクセスして、メイキングを検索した。
我这么感叹着，打开了JDA的技能数据库检索页，输入了「メイキング」。

% が、検索結果はやはり「なし」だった。\\
然后，果然还是「无结果」。\\

% 「おいおい、未登録スキルなのか？　これ」\\
「我靠，居然是个未登录技能啊这玩意？」\\

% ＪＤＡのデータベースは当然ＷＤＡに繋がっている。つまり世界中で今まで見つかったことがないスキルだってことだ。
自然，JDA的数据库是与WDA相通的。也就是说，这是至今为止尚未被发现的技能。

% 機能がはっきりしない未登録スキルは、販売チャンネルがほぼない。何しろ値段が付けられないからだ。調べている時間や交渉する時間も当然、ない。\\
搞不清楚效果的未登录技能，基本上是卖不出去的。毕竟没法定价，也自然没有去调查效果或者谈判的时间。\\

% 「まいったな……せっかく一攫千金をゲットして、会社も辞められると思ったのに」\\
「真是服了啊……我还以为能一夜暴富，辞职不干了呢」\\

% 俺はがっくりと肩を落として、明日の出社後に巻き込まれる面倒事を想像した。
我刷地放下了肩膀，大脑里开始想像起明天去公司后的麻烦事情。

% あまりの鬱な想像に、おもわず頭を左右に振って、お湯を沸かすために台所へと立った。
面对这过于令人郁闷的想像，我不禁甩了甩头，然后起身去厨房烧水。

% コンロにヤカンをかけながら、気分を変えようと、少し良いお茶を戸棚から取り出した。\\
我一边把烧水壶放在灶上，一边想着换换心情，从柜子里取出了稍微贵些的茶。\\

% 「星野村の玉露は、最高ですってね」\\
「星野村的玉露茶真是最棒啦」\\

% 少し沸騰させた後、コンロから下ろして、お湯の温度が下がるのを待っているあいだ、もう一度机の上のオーブに目をやった。\\
等水沸腾了一会之后，我把水壶拿了下来。等着热水凉一凉这会，我再次看向了桌上的宝珠。\\

% 「やっぱり自分で使うしか、ないか」\\
「果然只有自己用了啊」\\

% 八女の玉露を丁寧に入れて、カップに注いでコタツまで持って行くと、まずはそれを一口飲んだ。\\
我小心翼翼地向杯中放入八女玉露，然后将杯子缓缓端到被炉上，然后试嘬了一口。\\

% 「ん？　なんかいつもより旨味が強く感じるけど……なんかしたっけ？」\\
「嗯？总觉得旨味好像比以前更浓郁了……我有多做啥来着？」\\

% ま、うまいのならいいか、と特に深く考えず、オーブを手にとった。\\
嘛，好喝就好喝嘛，反正是好事——我不做多想，拿起了宝珠。\\

% 「やっぱりここはお約束だよな」\\
「果然这里要按惯例来一发呢」\\

% 俺はそう呟くと、小さく叫びながらオーブを使った。\\
我这么小声念到，然后一边控制着音量喊着一边使用了宝珠。\\

% 「おれは人間を辞めるぞ！」\\
「我不做人啦！\footnote{JOJO梗，不多解释了}」\\

% それは不思議な感覚だった。何かが体にしみ通ってくるような、体が一度バラバラにされて、再構成されていくような――気味は悪かったが気分は悪くなかった。\\
那感觉十分不可思议。像是什么东西通过了身体，又像是身体被一度分解，然后又被重新组合——虽然感觉很奇怪，但并不会感觉不快。\\

% 「ん……」\\
「嗯……」\\

% 俺は、目を開くと、手を握ったり開いたりしながら、その感触を確かめた。
我睁开了眼，握了握拳，确认了一下自己的感觉。

% 特に何かが変わった、といった感じはしない。\\
但并没有感觉到什么明显的变化。\\

% 世界が劇的に変わって見えるかもと考えていた俺は、少し落胆した。\\
我本以为眼中的世界会发生什么戏剧性的转变，现在则有点失望。\\

% 「ま、Ｈと同じで、経験してしまえば、どってことないのかもな。……スキルってどうやって使うんだ？」\\
「嘛，大概是跟做爱一样，经历过了就不当是什么大事了吧。……说起来技能该怎么用啊。\\

% 困ったときのインターネットだ。
有问题上网就好了。

% スキルを使ったヤツの体験談を検索した。もちろんそれが、嘘かホントかは判断のしようがないが。\\
我试着搜了搜使用技能的体验记。当然，里面写的是真是假我是没法判断的。\\

% 「なんだかなぁ。どれを読んでも、要約すれば『なんとなくわかる』だよ。なんとなく。なんとなくねぇ……」\\
「什么啊。读了一堆，说来说去不过是『不知不觉就会了』。不知不觉啊。不知不觉……」\\

% 目を閉じたり、腕を組んだり、残った茶を飲んだりしたが、結果は――\\
我试着闭起眼，叉起手，又喝完了剩下的茶，可结果——\\

% 「……さっぱりわからん」\\
「……完全搞不明白」\\

% だった。\\
行不通。\\

% もしかして、取得に失敗した？　そう考えた俺は、Ｄカードのことを思い出した。
难不成我是没有获得成功？我这么想到，然后想起了D卡的事情。

% 確か、Ｄカードには、取得スキルが記載されるはずだ。\\
确实，在D卡上面应该记载着获得的技能才是。\\

% 「って、カード何処に置いたっけな。確か、はいてたズボンのポケットに……」\\
「说起来我把卡放哪去了来着。好像是在那条裤子的口袋里……」\\

% 脱衣場のカゴからそれを取り出すと、ポケットをあさって、くすんだ銀色のカードを取り出した。\\
我从 脱衣间的篮子里找出那条裤子，摸了摸口袋，取出了那张银色的卡片。\\

% 「お、あった、あった。さて、スキルはっと……」\\
「哦，找到了找到了。我看看，技能技能……」\\

「啥？」\\

「排、排名、第一？」\\

「等等等等等等等等，我记得不是9900万左右来着！？」\\

可不论我看多少遍，上面写着的依然是第一。

如果按照假设，排名是按照打倒怪物后获得的经验值排出来的的话……」\\

% 「鉄筋を落とした後、しばらくして聞こえてきた、あの不気味な声、か？」\\
「难不成，是当时钢筋掉下去一会之后听见的，那个毛骨悚然的声音吗？」\\

「第一说不定很强呢？」\\

军队里那些人应该是从一开始就泡在迷宫里了吧。虽说现在也有一般的探险者了，但估计排在前头的那些人大多都是军队或者警察的人吧。

% ３年の経験をごぼう抜き？\\
刷地一下就超过了他们三年的经验？\\

说是这么说，可完全没有实感啊。\\

% 「だけど、強くなったって感じは全然しないんだよな。別段、ドアノブをひねりつぶせるわけでもないし」\\
「可跟我说我变强了我也完全没有感觉啊。刚开门的时候也没有咔嚓一下把把手给拧下来啊」\\

% 力一杯玄関のノブを握りしめてみたが、別になにも起こりはしなかった。\\
我试着用力握住玄关的把手，却也没有发生什么特别的事情。\\
