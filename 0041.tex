\subsection{§041 三代绘里 11/22 (四)}

% 「あ、大丈夫でしたか？！」\\
「啊，你没事嘛！？」\\

% 小川の向こうから、コンパウンドボウを構えていた女性が声をかけてきた。\\
在小河对面，手上架着复合弓的女性像我搭话。\\

% 「え？　ああ、まあなんとか」\\
「诶？啊，嘛想办法解决了」\\

% 俺は曖昧に笑うと、小川で手を洗った。\\
我暧昧地笑了笑，然后在小河里洗了洗手。\\

% 「それで、彼の容態は？」\\
「所以，他现在什么情况？」\\

% 川を飛び越して近づいてみれば、一緒にいた男の腕は、随分酷い状態のようで、脂汗を浮かべながら横になっていた。\\
我跳过小河，走进那两人，其中那个男的手臂伤势看起来十分严重，正躺在地上因疼痛满头大汗。\\

% 「一応止血だけはしたんですが……薬も道具もないので」\\
「姑且给他做了止血……可是我手上既没有药也没有道具」\\

% そう言って彼女は心配そうに、男を見ていた。彼氏なのかね？\\
女性一边说着，一边一脸担心地看着男性。是男朋友吗？\\

% しかし困ったな。５層の入り口付近にいる連中の所まで送って別れるつもりだったんだが……かといって、このまま拠点車に連れて行くのもためらわれる。
不过这可难办了啊。我本来是打算把他们送去5层入口附近那帮人那里的……不过也在犹豫要不要干脆把他们带去我们的据点车那。

% 一応、三好が用意した普通の救急セットもあるにはあるが、どうやら右の前腕をごっそりヘルハウンドに持って行かれてるっぽい。
虽说三好姑且也有准备好普通的急救工具箱，但看起来他整个右手上臂都被地狱犬给咬掉了。

% 医者ならすぐに肘の先で腕を落として治療しかねない大けがだ。普通の救急セットで間に合うとはとても思えなかった。\\
伤得这么重，要是医生的话说不定会直接让他截肢。怎么看普通的急救箱也应付不过来。\\

% 「……仕方ない」
「……没办法」

% 「え？」\\
「诶？」\\

% 三好、すまん。\\
三好，抱歉。\\

% 「使いますか、これ？」\\
「这个，要不要？」\\

% 俺はバックパックからを装って、さっき手に入れたポーションを取り出した。
我装作伸进包里，然后掏出了刚刚拿到的药水。

% それに触れた女は、思わず声を上げていた。\\
而触碰了药水之后，女性下意识地出了声。\\

% 「え？　ええ？！　ヒールポーション？！　しかもランク５！！」\\
「诶？呜哇！？治疗药水？！还是等级5的！！」\\

% しまった、ポーションのランクなんかチェックしていない。ランク５って凄いのか？
糟了，药水等级忘了看了。等级5很厉害么？

% 「ランク５！」と三好が叫ぶ声がイヤープラグ型のデバイスから聞こえてくる。\\
「等级5！」只听见三好的喊声从耳麦中传出来。\\

% 「こ、これ……でも……」\\
「这、这……可是……」\\

% 彼女はポーションと、男を交互に見ながら、葛藤していた。\\
她来回看着手上的药水和男性，一脸纠结。\\

% 「早く使わないと拙いんじゃないですか？」\\
「不赶紧的话不是不大妙吗？」\\

% 俺がそう言うと彼女は覚悟を決めたような顔をした。\\
我这么问到，然后她便露出一副下定了决心的表情。\\

% 「ありがとうございます。必ずお支払いします」\\
「十分感谢。我一定会把钱还上的」\\

% そう言って、男に駆け寄ると、ポーションを少しずつのませている。\\
她这么说完，然后跑向男性，让他慢慢地喝下了药水。\\

% あー、やっぱり飲むものなんだ。患部にかけてもいいのかな？　なんて考えていると、大きなあくびがひとつ出た。
啊，果然是用喝的啊这。我一边考虑着撒在患部是不是有效这一问题，一边深了一个大大的懒腰。

% 超回復を持っていても、緊張や集中をしていないと眠気は襲ってくるようだ。そうでなきゃ不眠症になるからだろう。\\
就算有超回复辅助，但在情势不紧张、精神不集中的时候还是会困的。要不然就成失眠症了。\\

% 相も変わらずダンジョンアイテムの効果は劇的だった。
迷宫物品的效果一如既往地立竿见影。

% アーシャの超回復の時ほどではないにしろ、筋肉はおろか、骨まで欠けていた前腕の内側が、みるみる盛り上がって元に戻っていく様はまさに圧巻と言える。\\
尽管到不了爱夏的超回复那时的程度，但那不仅是肉，甚至连骨头都缺掉一块的上臂内侧慢慢长回去的情景，也实在是令人惊叹。\\

% ポーションを全て飲み終わる頃には、男は完全に回復していた。\\
把药水全部喝光之后，男性身上的伤完全消失了。\\

% 「え？　あれ？　姉ちゃん？」\\
「诶？啊？姐？」\\

% もうろうとしていた男は、意識がはっきりすると、不思議そうに自分の腕を眺めて言った。
刚刚还在意识朦胧的男性，意识总算清醒过来，然后一脸不可思议地看着自己的手问到。

% なんだ、姉弟だったのか。\\
什么嘛，原来是姐弟啊。\\

% 「翔太！」\\
「翔太！」\\

% 女は涙を浮かべて弟に抱きついた。仲いいな。\\
女性一边哭着一边抱住了弟弟。关系真是好呢。\\

% 「一体何が……俺の腕……ついてる」
「到底怎么回事……我的手……接回去了」

% 「あの人が……」\\
「那边那个人……」\\

% そう言って彼女はあったことを弟に説明していた。\\
然后她便向弟弟解释了一边刚才发生的事情。\\

% 「ランク５？！」\\
「等级5？！」\\

% 黙って話を聞いていた弟が、驚いたような声を上げると、俺の方を厳しい眼差しで射貫いてきた。
刚才一直默默听着的弟弟，突然吓了一跳般喊出了声，然后一脸敌意地瞪住了我。

% え？　ここは感謝される場面じゃないの？\\
诶？搞啥，居然不谢我？\\

% 「俺は助けてくれなんて言ってねぇし。別に大した怪我もしてなかったし」
「我也没说要你来帮我。反正我也没受什么大伤」

% 「し、翔太？」\\
「翔、翔太？」\\

% は？　いきなりなにを言い出すんだ、こいつ？\\
啥？这小子突然发什么癫？\\

% 「大体ポーションを使ったなんて証拠はないし」
「反正你也没有用过药水的证据」

% 「ちょっと、あなた！　何を言い出すの！」\\
「等等，翔太！你说什么呢！」\\

% うんまあ、そうかも。厳密に言えばあるのかも知れないけれど、俺はしらん。\\
嗯嘛，或许吧。严格来说说不定是有的，不过我是不大清楚啦。\\

% 「ご、ごめんなさい。うちの弟、錯乱しちゃってて」
「实、实在抱歉。我弟弟现在有点搞不清情况」

% 「謝ることなんかないよ！　姉ちゃんは騙されてるんだ！」\\
「道什么歉啊！是姐你被骗了啊！」\\

% はい？\\
搞啥？\\

% 「そのヒヒジジイは、ランク１でも充分なところにランク５なんてポーションを持ち出して、姉ちゃんを借金漬けにして自由にしようと目論んでるんだよ！」
「明明等级1的药水就够用了，他却拿出等级5的药水来，那个猴子大叔是打算让姐你欠他钱不安好心啊！」

% 「翔太！」\\
「翔太！」\\

% おお！　それは初耳、お釈迦様も吃驚だぜ。ああ、もう面倒くせぇな。
哎唷！真是长见识了，恐怕释迦牟尼听了都要吃惊吧这。啊啊，真是摊上麻烦事了。

% イヤープラグからは、「一応録画してありますけど」といいながら、三好が肩をふるわせて笑いをこらえている雰囲気が、ありありと伝わってくる。\\
我戴着的耳麦里，传来了三好「我们姑且是有录像的」的声音。我好像能看见她那抖着肩膀憋笑的样子了。\\

% 「あの、もういいですから。そっちへ行くとすぐに４層への上り階段です。何組かのチームが野営してますから、そこで朝まで過ごして帰られるといいですよ」
「那个。我无所谓了啦。往那边走马上就是到4楼的楼梯了。那边应该有几个队伍在野营，你们就去那边等天亮再回去吧。

% 「え？」
「诶？」

% 「いや、ですから……」
「哎呀，所以说啦……」

% 「ほっとけよ！　いいって言ってんだからさ！　いくぜ、姉ちゃん。そういや、坂井と当麻は？」\\
「管他呢！他不是说了能走了吗！姐，赶紧走吧。说起来，坂井和当麻呢？」\\

% ああ、あの逃げ出したやつらか。\\
啊啊，那俩溜了的家伙啊。\\

% 「逃げたみたいですから、同じ場所に居るんじゃないですか？」
「他们好像都先跑了，大概也在那个地方吧？」

% 「なら、すぐに行こうぜ！」\\
「那就赶紧走去找他们吧！」\\

% お前達姉弟を餌にして逃げたヤツラだけどな。それを知っている姉の方は、とても複雑な顔をしていた。\\
虽然那几个家伙可是拿你们姐弟当七子自己开溜了啊。而知道这回事的姐姐，脸上则露出了十分尴尬的表情。\\

% 「これ、私の連絡先です。必ずお支払いしますので……失礼ですがお名前を」
「这是我的联系方式。钱我一定会付上的……冒昧问下您叫」

% 「別に名乗るほどのことでは。文字通り犬にでも噛まれたと思って忘れますよ」
「没什么大不了的。你就当字面上地被狗咬了就行，不用上心」

% 「そんな……」\\
「我才不会……」\\

% 泣きそうな顔を向けてきた彼女は、基本的に善人なんだろう。
她一脸想要哭的表情，基本上应该是好人吧。

% 別に優しくすることだって出来ただろうが、ヒヒジジイ扱いされたら多少は腹も立つ。彼女のせいではないってことは、よく分かっているけれども。\\
其实我也可以温柔一点，但毕竟被叫成猴子大叔，心里多少还是有些不愉快的。虽然我也清楚这不是她的错。\\

% 「なにやってんだよ、姉ちゃん！　早く行こうぜ！」
「姐你还磨蹭啥啊！赶紧走啊！」

% 「大丈夫。きついことを言って申し訳ない。ほら、行ってください。きっとまたどこかで会えますよ」
「没关系。抱歉刚才说那种话。快走吧。以后肯定还会再见到的啦」

% 「すみません。できれば連絡を下さいね」\\
「实在是抱歉。可以的话到时候请联系我」\\

% そういって、彼女は弟の後を追いかけていった。\\
她这么说完，然后追着她弟弟离开了。\\

% 「三代(みしろ)絵里(えり)、ね」\\
「\ruby{三代|绘里}{みしろ|えり}，么」\\

% 俺は貰った連絡先を仕舞いながら、拠点車へ向かい、彼女たちとは反対の方向へと歩いていった。\\
我一边收起收到的名片，一边转向与他们相反的方向，朝着据点车走去。\\

\psline

她再登场就是等很久之后了。
