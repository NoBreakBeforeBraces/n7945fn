\subsection{§050 彷徨之馆（后篇）}

% そこはただのガランとした広い部屋だった。\\
眼前，是一间空空荡荡的宽广屋子。\\

% 「普通こういう大邸宅の玄関を入ったら、そこはエントランスホールで、２階へ上がるダブルサーキュラー階段とかあるんじゃないですかね？」
「一般来说这种大宅子进了门之后，前面就应该是一个大厅，然后两边应该有两条弧形楼梯的不是嘛？」

% 「サーキュラー階段ってなんだ？」
「弧形楼梯是什么？」

% 「ぐるって廻るようなデザインの階段です」
「就是那种弯着转半圈走上去的台阶」

% 「あー、映画に出てくる大豪邸にありそうなやつか」\\
「啊，那种电影里的豪宅常有的那东西是吧」\\

% 俺はまわりを見回した。\\
我环顾了下四周。\\

% それは何の変哲もない、石造りで天井の高い部屋だった。ただし広い。３０ｍｘ３０ｍくらいはありそうだ。
眼前的，不过是一个平淡无奇的石质高顶屋子。总之就是大。感觉尺寸可能有30mx30m那么大。

% 壁には、書架が据え付けられ、入り口からはよく見えないが、びっしりと本が詰まっているようにも思えた。\\
墙壁上嵌着书架，虽然站在入口看不大清，但感觉好像摆满了书。\\

% 部屋の四隅には、いかにも中央に進むと動き始めますよと言わんばかりの彫像がすえられている。ノートルダム寺院のグロテスクのようなデザインだ。\\
屋子四角，各摆着一尊姿态蓄势待发，随时准备着冲向中央的雕像。跟巴黎圣母院的那设计一样诡异。\\

% 「あれもガーゴイルの一種かな？」
「那也属于石像鬼嘛？」

% 「ゲームなら部屋の真ん中に行くと動き始めそうですよね。ここはやはりあらかじめ壊しておきましょうか？」
「要是游戏的话，等我们走到屋子中间它们肯定就都会动起来了呢。要不我们稳妥点，先破坏掉它们？」

% 「普通、動き始めるまでは破壊不能アイテムじゃないか？」
「一般来说在动起来之前不都应该是不可破坏物品的嘛？」

% 「試すのはタダですよ」\\
「反正就试试，也没啥嘛」\\

% 三好がそう言ったとたん、４隅の彫像が吹き飛んだ。あの威力は、２ｋｇバージョンだな\\
三好话音刚落，四尊雕像应声被打飞了。看这威力，估计用的是2kg的那种啊。\\

% 「あれ？　もしかして、ただの大理石像だったのかも……」\\
「哦呀？难不成，那只是普通的大理石像嘛……」\\

% そう三好がテヘッった瞬間、後ろの扉が激しい音を立ててしまった。\\
正在三好卖傻的时候，身后的门突然发出了尖锐的声音。\\

% 「あっちゃー、もしかして怒らせましたかね？」
「哦吼，怕不是惹到家主人了？」

% 「他人の家に入って、いきなり玄関で彫像をぶちこわしたら、普通怒ると思うぞ」\\
「跑进别人家里，在门口突然把家里的雕像砸个稀烂，一般人都会生气的啦」\\

% まわりを警戒しながら入り口の方へ下がると、部屋の中央に３つの魔法陣が現れ、何かがそこからはい上がってきた。\\
我们一边提防着四周，一边退到入口处，便看见屋子中央出现了三个魔法阵，然后什么东西从里面浮现了出来。\\

% 「スケルタル・エクスキューショナー？！」\\
「\ruby{骷髅}{Skeletal}·\ruby{行刑者}{Executioner}！？」\\

% 代々木ではおそらく初発見だろう。巨大な剣を引きずって歩く大型のスケルトンだ。
在代代木出现这东西估计这是第一次。我们眼前，出现了三只拖着巨剑走动着的大型骷髅。

% 普段の動きはそうでもないが、攻撃時はその剣を振り回して突撃してくるらしい。\\
和平时不同，在攻击时它们会甩起手上的剑突击。\\

% 「先手必勝ってヤツだな」\\
「总之先发制人吧」\\

% 俺はいつものようにウォーターランスを発動して、３体のモンスターを砕いた……つもりだった。\\
我一如往常，放出了水长枪，然后这三只怪物便……并没有如我所望被击倒。\\

% 「おお？！」\\
「哦呀？！」\\

% 高速で発射された水の槍は、モンスターの前にある不可視のバリアのようなもので遮られ、霧散させられた。\\
高速射出的水流形成的枪，被怪物身前一个像是不可见的屏障般的东西挡了下来，消失掉了。\\

% 「先輩、鉄球をお願いします！」\\
「前辈，用铁球！」\\

% 俺は８ｃｍの鉄球を取り出すと、全力で一番手前のモンスターに投擲した。
我取出8cm大小的铁球，全力朝着最前面的那只怪物扔去。

% ガンという大きな音と共に、モンスターがのけぞる。これも耐えるのかよ！　ただ、魔法よりは効果がありそうだ。\\
随着咚的一声巨响，怪物的身子向后倒去。这都扛得住啊！不过倒是比魔法更有效的样子。\\

% 繰り返してやればいいかと、もう一度投擲する。その鉄球が当たると同時に、そいつの膝が砕けた。\\
我想着多来几下就行了，便再扔了一个球。然后正当我扔出的铁球砸中骷髅时，那家伙的膝盖突然碎裂开来。\\

% 「へ？」
「诶？」

% 「２カ所以上に同時攻撃すると、抗力が分散するみたいですね！」\\
「如果同时攻击两个地方的话，好像防御力就会被分散掉哦！」\\

% 俺が頭に対して投擲した瞬間に、膝を狙って鉄球を射出したらしい三好が言った。\\
三好这么说到。在我朝着骷髅头扔出铁球的同时，她好像也瞄准了膝盖将铁球射了出去。\\

% 「やるじゃん三好！　俺の後ろに隠れてなければ、格好いいぞ」
「挺能干的嘛三好！要是没躲在我身后的话可是很帅气的哦」

% 「何言ってんですか。盾は先輩が持ってるんですから、これでいいんです！」\\
「说什么傻话呢。毕竟盾是前辈拿着的嘛，这样就好这样就好！」\\

% まあ、そうかも知れないが、それじゃ他のヤツにも……\\
嘛，就当是这么一回事吧，但这么一来别的家伙也如法炮制……\\

% 「ああ！！　し、しまったー！？」
「啊啊！！糟、糟啦！？」

% 「え？！　どうしたんです！？」\\
「诶？！怎么了吗！？」\\

% 突然声を上げた俺に、驚いたような顔で三好が尋ねた。\\
听见我突然喊出了声，三好也紧张起来，向我问到。\\

% 「いや、せっかくボスっぽいモンスターなのに、雑魚のお供がいないから数が合わせられないんだよ！」
「你想，好不容易撞上个像是BOSS的怪物，结果身边没有小怪，没法刷杀敌数啊！」

% 「……先輩、意外と余裕ですね」\\
「……前辈，感觉你意外地从容呢」\\

% 他のモンスターにも、鉄球を飛ばして牽制しながら、三好が呆れたように言った。
三好一边用铁球牵制着其他怪物的行动，一边傻眼地说道。

% 結構固くて、とどめを刺すのに時間が掛かったが、三好と連係して数分で相手の機動力を削いだ後、力業で排除したのだった。\\
虽然对手硬的要死，明明都打中了要害却还是不能一击毙命，但我和三好一起花了几分钟消耗掉了对方的运动能力后，还是凭蛮力解决掉了。\\

「这回又换\ruby{托尔金}{Tolkien}了嘛？还挺博识的嘛，这迷宫。那难不成这个别墅其实是坟墓，而这把剑则是\ruby{佛罗多}{Frodo}的剑？」\\

「我还是第一次见掉落武器的诶。而且探索者指南随附的武器目录里也没有找到对应的条目……说起佛罗多的剑，那就是\ruby{刺针}{Sting}咯」

「那是他从\ruby{比尔博}{Bilbo}那拿到的剑啦。他在\ruby{古墓岗}{Barrow-downs}拿到的剑，是被\ruby{安格马巫王}{Witch-king of Angmar}弄断，在\ruby{瑞文戴尔}{Rivendell}忘了拿去修，最后还被刺针顶替了位置的可怜的剑哦」

% 「いや、可哀想て……」\\
「不是我说，可怜这……」\\

% "Scimitar of Deserts" かな？　塚山なら砂だらけの印象だからおかしくはないけど、ここは館だしなぁ。\\

「前辈，看那个！」\\

% 俺が全てのアイテムを拾い終わると同時に、部屋の中央に何かが現れた。\\
正在我把所有的东西都收纳起来了的时候，屋子中央突然出现了个什么东西。\\

% 「碑文？」\\
「碑文？」\\

% 現れた台座の上には、随分立派な装飾が施された本のページのようなものが置かれていた。そうして、台座には、門柱にあったのと似たような文字で何かが書かれていた。
在出现的基座上，立着一个装饰的颇为豪华的如同书本的一页般的东西，而基座上，则以与门柱上相似的文字写着些什么东西。

% 俺はスマホでそれを撮影した後、台座の中央に置かれた本のページのようなものをまじまじと眺めた。\\
我用手机拍完照之后，开始仔细地观察起基座中央摆着的如同书本的一页般的东西。\\

% てくてくと三好が書架に近づいていく。\\
三好悠闲地往书架走去。\\

% 「あんまりウロウロするなよ、罠とかあるかも知れない――」\\
「你别瞎晃悠啊，说不定有什么陷阱——」\\

正在我这么说着，拿起碑文的那一瞬间。\\

尖塔上突然响起钟声，然后一股如同屋子所在的空间开始扭曲般的诡异感突然袭来。\\

我一边用手上拿着的铁球迎击袭来的大乌鸦，一边让三好先跑，自己则架起盾殿后，放出了水长枪。\\

虽然石像鬼们哪怕缺了翅膀少了脚，甚至头都没了，也还在凭着势头朝我冲来，但我还是依靠自己的高状态值巧妙地躲开了他们的攻击，用盾把他们拍了下来。
